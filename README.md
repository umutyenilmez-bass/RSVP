# Davetiye & RSVP

Kişiselleştirilmiş WhatsApp davet ve RSVP takip uygulaması.

## Özellikler

- Etkinlik detayları (ad, tarih, yer, program) girişi
- Davetli listesi yönetimi (tek tek veya toplu ekleme)
- Her davetliye özel kişiselleştirilmiş WhatsApp mesajı
- Tek tıkla "Geliyorum / Gelemiyorum" RSVP sayfası
- Anlık katılım takip panosu

## Kullanım

1. **Etkinlik** sekmesinden etkinlik bilgilerini gir
2. **Davetliler** sekmesinden kişileri ekle
3. Her kişi için WhatsApp butonuna bas — mesaj hazır açılır, gönder
4. **Takip** sekmesinden kimlerin gelip gelmeyeceğini görüntüle
